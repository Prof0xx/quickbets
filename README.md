# QuickBets

> Prediction market on Base with quick fulfillment (15-minute durations) for social events and esports.

## Overview

QuickBets is a fast-paced prediction market platform built on Base blockchain, designed for quick betting experiences on social events and esports with rapid 15-minute fulfillment cycles.

## Tech Stack

- **Framework:** Next.js 14+ (App Router), TypeScript
- **Database:** Prisma + Neon (Postgres)
- **Authentication:** Clerk
- **API:** tRPC
- **Styling:** Tailwind CSS
- **Monitoring:** Sentry
- **Caching:** Upstash
- **Testing:** Playwright
